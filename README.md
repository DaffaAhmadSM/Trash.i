# Trash.I

Trash.I is a Vite + React + TypeScript frontend for a waste pickup service.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

Install dependencies and start the dev server:

### Using pnpm

```sh
pnpm install
pnpm dev
```

### Using npm

```sh
npm install
npm run dev
```

The app will be available at the URL printed in your terminal (typically `http://localhost:5173`).

## Build

Create a production build:

### Using pnpm

```sh
pnpm build
```

### Using npm

```sh
npm run build
```

## Preview Production Build

Preview the production build locally:

### Using pnpm

```sh
pnpm preview
```

### Using npm

```sh
npm run preview
```

## Lint

Run ESLint:

### Using pnpm

```sh
pnpm lint
```

### Using npm

```sh
npm run lint
```

## Routes

- `/` — Home
- `/login` — Login
- `/register` — Register
- `/book-pickup` — Book Pickup
- `/checkout` — Checkout
- `/waste-categories` — Waste category details

## Notes

- The UI is based on the Figma designs provided by the project.
- This is a frontend-only demo; authentication and booking actions are mocked.
